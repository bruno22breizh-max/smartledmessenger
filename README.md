# Smart Led Messenger

Intégration Home Assistant pour contrôler vos Smart Led Messenger :
- Envoyer des messages texte
- Réglage de l’intensité et de la vitesse
- Presets (Urgent / Info / Nuit)
- Détection online/offline
- Support multi-appareils
- Entités `text`, `number`, `button`
- Prêt pour HACS
